Rocket Repo

The Rocket Repo is a repository which contains diagrams and figures of prominent spacecraft designs. The goal of this page is to provide some basic info about these crafts as well as how they work. Additionally, the site serves as a launch ground to test website design, security, and functionality.

While this project is specifically for IT&C 210 which looks at web systems and security, it has ben decided that the addition of rockets are nessasary because they are cool.

The Space Race has many ties to security. Whether it was the fear that the Space Shuttle might be used as a bomber or the constant exchange of intelligence between the world's superpowers, this era has a lot we can learn from.
